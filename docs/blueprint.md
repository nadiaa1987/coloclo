# **App Name**: LinkVision

## Core Features:

- Image Generation from Link: Generates an image from a prompt sent to the following URL: https://gen.pollinations.ai/image/{prompt}?model=flux&width=1024&height=1024&seed={random}&key=sk_UOsZfgtGMSYNskyHUmwbWTQEdYPKv2UxR
- Prompt Input: A simple text box will take the prompt from the user. This text prompt will be sent as part of the URL for image generation.
- Image Display: Displays the generated image.

## Style Guidelines:

- Primary color: Electric Indigo (#6F00FF) to evoke creativity and digital innovation.
- Background color: Light gray (#F0F0F0) providing a clean, neutral backdrop.
- Accent color: Bright Cyan (#00FFFF) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look, suitable for all text.
- Use minimalist icons for controls (e.g., refresh, download).
- Simple layout with the prompt input at the top, followed by the generated image.
- Subtle loading animation while the image is being generated.
